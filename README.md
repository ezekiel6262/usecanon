# Canon marketing homepage

Static Vercel project for `usecanon.xyz`.

- `Open Studio` links to `https://app.usecanon.xyz/`.
- `Generate next episode` links to `https://app.usecanon.xyz/#episodes`.
- `Docs` opens the standalone documentation experience at `/docs/`.
- No API keys or runtime environment variables are required.

Deploy this directory as a separate Vercel project, then attach `usecanon.xyz` and `www.usecanon.xyz` in Project Settings → Domains.
